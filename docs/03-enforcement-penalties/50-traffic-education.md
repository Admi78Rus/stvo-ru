# § 50 Специальные правила для острова Гелиголанд

:::info[Оригинал]

[§ 50 Verkehrsunterricht](https://www.gesetze-im-internet.de/stvo_2013/__50.html)

:::
На острове Гелиголанд запрещено автомобильное и велосипедное движение.
